# Creepypasta Abyss — Fabric 1.21.4

Полный проект Fabric-мода под **Minecraft 1.21.4**. Подготовлен для сборки через GitHub Actions.

## Что реально изменено

- фото из загрузки стоит как **скример** и как **скин сущности**;
- три загруженных MP3 конвертированы в OGG и подключены как:
  - `entity.haunter.screamer` — крик скримера;
  - `entity.haunter.breath` — дыхание сущности;
  - `entity.haunter.ambient` — мрачный фон/стон;
- сущность `creepypasta:haunter`:
  - без ника;
  - сначала стоит и смотрит;
  - если игрок её видит — резко активируется;
  - может телепортироваться ближе;
  - преследует игрока;
  - ломает блоки перед собой;
  - при касании включает скример и телепортирует в измерение;
- скримеры:
  - не постоянные;
  - редкие рандомные;
  - срабатывают при касании сущности;
  - срабатывают при резком обнаружении сущности;
- заражение мира идёт **постепенно**, а не сразу;
- постепенно ломаются блоки, листья, куски чанков;
- молнии, сбои времени, день/ночь;
- жуткие сообщения в actionbar/чате без текста `ОНО КОСНУЛОСЬ ТЕБЯ`;
- таблички `HELP!!!`;
- измерение `creepypasta:nightmare`:
  - большое;
  - тёмное;
  - мёртвые деревья без листвы;
  - кресты;
  - портал обратно;
  - около портала остаётся только одна сущность-охранник.

## Сборка через GitHub

1. Создай пустой GitHub-репозиторий.
2. Загрузи туда все файлы из архива.
3. Открой вкладку **Actions**.
4. Запусти workflow **Build Fabric Mod**.
5. `.jar` появится в артефактах workflow.

## Локальная сборка

Нужен Java 21 и Gradle.

```bash
gradle build --no-daemon
```

Готовый jar будет здесь:

```text
build/libs/creepypasta-abyss-1.0.0.jar
```
